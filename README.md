# Django CMS Example

### Create a virtualenv

```shell
$ virtualenv env
$ source env/bin/activate
```

### update pip

```shell
$ pip install --upgrade pip
```

### install django-cmsinstaller

```shell
$ pip install djangocms-installer
```

### Run the migrations

```shell
$ python manage.py makemigrations
```

### Run application

```shell
$ python manage.py runserver
```
